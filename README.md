## 🔧 How to Use genlayer-anime-trivia

Once the game is installed, start it from your desktop or search for "genlayer-anime-trivia" in the Start menu.

### Creating an Account

The first time you open the game, you will be asked to create an account. This account links to your GenLayer wallet, which holds your tokens.

- Enter a username of your choice.
- Set a strong password.
- Confirm your password.
- Follow any prompts to create or import a GenLayer wallet. If you are new, the game will guide you through generating a secure wallet.

### Joining a Match

1. After login, find the "Play" button on the main menu.

2. Choose "1v1 Duel" mode.

3. Set the token wager amount you wish to stake.

4. Confirm your wager to enter the queue.

5. The system will find another player with a similar wager.

### Playing the Game

- You will see one trivia question about anime at a time.
- Each question has multiple choice answers.
- Select your answer before the timer runs out.
- Points accumulate with every correct answer.
- The first player to reach the set score or highest score after all questions wins the match.

## 🛠 Troubleshooting and Tips

- **The game won’t launch**: Make sure your Windows version is up to date and you installed the required .NET Framework.

- **Problems connecting to GenLayer blockchain**: Ensure you have an active internet connection. Try restarting your router if issues persist.

- **Lost tokens or wallet access**: Keep your wallet seed phrase safe. If you lose it, you may lose access to your tokens permanently.

- **Game freezes or crashes**: Close other programs running in the background. Check if your graphics drivers are updated.

- **Questions seem repeated or outdated**: The game updates trivia questions regularly with each release. Make sure you download the latest version.
